⏰ Cita DNI Watcher

Bot que monitoriza la disponibilidad de citas para el DNI en España y te avisa por Telegram en cuanto se libera un hueco — sin que tengas que estar pendiente de la página.

⭐ Ver en GitHub 📦 Descargar ZIP

🎯 El problema

Renovar el DNI en España implica esperar más de dos meses para conseguir cita, y no existe ningún sistema oficial que te avise cuando hay huecos disponibles.

Cita DNI Watcher monitoriza la página por ti y te notifica por Telegram en el momento exacto en que se libera un hueco, sin que tengas que estar pendiente.

⚙️ Cómo funciona

1
Abre un navegador con Playwright Haces el login manual una vez y el bot se encarga del resto.
2
Monitoriza la página cada N minutos Detecta cambios y extrae los meses disponibles automáticamente.
3
Te avisa por Telegram Mensaje + captura de pantalla cuando aparece una cita dentro de tu rango aceptable.

🚀 Inicio rápido

$ git clone https://github.com/rubencougil/cita-dni-watcher.git
$ cd cita-dni-watcher
$ npm install && npx playwright install chromium
$ cp .env.example .env  # añade tu token de Telegram y chat ID
$ npm start
